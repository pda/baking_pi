Baking Pi
=========

ARMv6 GNU assembler code for [Raspberry Pi][1], based on the [Baking Pi:
Operating Systems Development][2] course by Alex Chadwick.

Developed using the [YAGARTO][3] GNU development toolchain for ARM.
